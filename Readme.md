Hospital Patient Data Analysis Using SQL
A SQL-based healthcare data analysis project demonstrating how SQL can be used to extract meaningful business insights from hospital patient records.

Project Overview
Healthcare organizations generate large volumes of operational data every day. This project analyzes a hospital dataset containing 200 patient records to explore patient demographics, billing patterns, diagnosis trends, ward utilization, and hospital revenue. The analysis answers practical business questions using SQL and demonstrates techniques commonly used in data analytics.
Project Objectives
•	Determine the overall size and characteristics of the patient population.
•	Analyze demographic patterns such as age and gender.
•	Identify cities contributing the highest number of patients and revenue.
•	Compare diagnoses and hospital wards using aggregate analysis.
•	Categorize patients based on treatment costs.
•	Identify high-cost patients using subqueries and Common Table Expressions (CTEs).
•	Generate business insights to support hospital management.

SQL Concepts Used
SELECT, WHERE, Aggregate Functions, GROUP BY, HAVING, ORDER BY, CASE Statements, Subqueries, and Common Table Expressions (CTEs).

Business Questions Addressed
1.	How many patients are in the hospital?
2.	What is the average age of patients?
3.	What is the gender distribution?
4.	Which city has the most patients?
5.	Which diagnosis has the highest average bill?
6.	Which wards have more than 20 patients?
7.	How can patients be categorized based on their bill amount?
8.	Which patients have bills above the overall average?
9.	Which patients belong to the high-bill category using a CTE?
10.	Which city generates the highest hospital revenue?
11.	Who are the top 10 highest-billed patients?
12.	Which age group generates the most revenue?
Key Findings
•	200 patient records were analyzed.
•	The average patient age is 48.46 years.
•	Male patients slightly outnumber female patients.
•	Islamabad has the highest patient volume and generates the highest hospital revenue.
•	Infection has the highest average treatment cost.
•	The General Ward has the highest patient load.
•	Patients aged 51 years and above generate the highest hospital revenue.
